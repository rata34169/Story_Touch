軟體開啟方式:下列兩方法擇一

package匯入方式:
	Asset -> Import package -> Custom package...（瀏覽:Story_Touch.unitypackage）

Unity開啟方式:
	打開Unity hub -> 舊專案 -> 瀏覽介面:Story_touch_PC_w_VR


＊現版本的Story_touch已將控制指令全數轉移至VR平台，使用時請設定好VR裝備
＊本專案使用的VR裝備為：Oculus Rift S
